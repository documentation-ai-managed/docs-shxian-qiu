## Create a New Project

<Steps>
  <Step title="Go to dashboard">
    ````
    Visit https://shxianqiu.com and log in.
    ````
  </Step>
  <Step title="Start new project">
    ````
    Click "Create New Project" and fill out the project info.
    ````
  </Step>
  <Step title="Add documentation">
    ````
    Use the rich-text editor to create your first docs page and click Save.
    ````
  </Step>
</Steps>
