---
title: Cannot Find Documentation Page
description: Resolve missing doc pages in shxian qiu.
---

## Symptoms

- Clicking on a doc link shows "Page Not Found"
- Newly created docs do not appear in the sidebar

## Solution

<Steps>
  <Step title="Refresh navigation">
    ````
    Click the reload icon at the top of the sidebar to update the doc list.
    ````
  </Step>
  <Step title="Check permissions">
    ````
    Make sure you have access to the project and its docs settings.
    ````
  </Step>
</Steps>
